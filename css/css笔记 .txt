一、理解web标准的含义---为什么采用web标准

*******将内容与样式分离

1、web标准时一系列标准，一系列技术标准组合应用。

【1】、结构化内容 xthml
【2】、表现化内容 css
【3】、行为化内容 javascript

2、网页开发的小工具---火狐浏览器中的firebug

作为web前端工作：要兼容的浏览器，IE/6/7/8/9,火狐浏览器，谷歌浏览器，苹果浏览器等。

网页是否有css样式，测试--浏览器改变文字大小。

3、css定义的技巧：
【1】、为了将来的css代码优化，建议所有属性后面都要加上“；”
【2】、某些xhtml标签都有自己默认的css属性值
	例如h1标签九幽自己的属性值，自动加粗显示，字号比较大。
【3】、为了兼容主流浏览器，为同意，建议我们在设置时，将所有的元素css属性，重置为标准的。
【4】、不同的浏览器，各自有不同的css属性值----要命的就是浏览器的兼容性
【5】、就一个比较特殊的字体想用，怎么办？--使用图片格式
	分中英文字体，设置字体的顺序，将英文字体设置在前，中文在后。
【6】、css编辑工具---用什么都行。
【7】、我们使用95%的css，都是常用的属性
【8】、想让文字在垂直方向上居中，将行高设置为元素的高度，行高=元素高度
【9】、css可以重新设置，xhtml标签的默认样式。

4、css应用网页的方式：就相当于，给人怎么去穿上衣服。

【1】、行内样式：应用于一个
【2】、嵌入样式，应用于一类标签
【3】、如果我们有很多网页，都要css，将这些css写到一个独立文件中，这个就是外链式，推荐使用
【4】、导入式样式：也是讲样式写到一个文件中，再导入网页中
	*******说明，导入式其实与内嵌式是相类似的，都是讲样式加入到网页里：
		1、导入时，会占用html文件空间
		2、有些浏览器，解析会有问题，浏览器会最后读取@import中的内容。




css作用就是修饰网页样式的---xhtml控制网页结构--给他穿上衣服的就是css

css应用于网页的四种方式：
1、行内样式：直接写在xhtml文档内，作用于标签内，只针对某一个标签，不灵活；
2、内嵌样式：直接写在xhtml文档内，可以针对某些标签
3、外部链接式：可以单独写在一个css文件中，然后去应用给要使用样式的html文件。
4、导入方式：可能有解问题，不建议使用。


我们想针对网页中的某一个元素或者一类元素设置样式--可能通过不同的css选择符


二、css选择符
1、什么是选择符？就是你要控制的对象
2、选择符的分类：
【1】、标签选择符---针对html标签
【2】、id选择符---针对某一个id
【3】、类选择符---针对某一类、或者某一些元素，他们的样式是相同的。
*******同一个元素，用用于多个类，类名与类名直接用空格分隔class=“类1 类2”
*****类名或id名，不要以数字开头。

三、css的特性
css叫层叠样式表

1、继承：子元素会继承父元素的某些样式。【因为有些子元素本身就有默认值，所以它不用父元素】

2、层叠：子元素如果定义了与父元素相同的样式，会覆盖掉父元素的样式
   *****后面的样式，会覆盖前面的样式。

四、css样式的优先权：当样式有冲突时，听谁？

css优先权：就近原则
作用范围越小，优先权越高。
需要修饰目标越近的样式优先权越高。

选择符优先权：
行内>id>class>标签选择符

特殊情况，我们可以提升某个属性的权限。
